## How to use

1. Read the draft end-to-end.
2. Fill in every `{{TODO}}` placeholder with a personal detail. These are the difference between "AI-written content" and "trustworthy first-person account."
3. Edit freely. The drafts are scaffolding, not scripture.
4. Check the editing notes at the bottom of each file — they flag voice, timing, hashtag strategy, and common mistakes to avoid.
5. Schedule or publish per the target dates above.
