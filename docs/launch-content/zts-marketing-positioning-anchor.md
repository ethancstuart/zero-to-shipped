# ZTS Marketing — Positioning Anchor

**For:** zts-marketing.vercel.app/config → "Positioning Anchor" field
**Format:** 2 sentences — core claim + core enemy

---

## Positioning Anchor

Zero to Ship teaches PMs, BAs, and project managers to build real, deployed software with AI coding tools — using the analytical and specification skills they already have. The enemy is the belief that shipping software is for engineers only, and the default that non-engineers keep waiting for their prototype to be prioritized.

## Notes

- The cron uses this anchor to maintain brand voice across all generated posts
- "Core claim" = what ZTS does and for whom (with the insight that existing PM skills transfer)
- "Core enemy" = the problem you're solving (both the belief AND the practical backlog bottleneck)
- Edit this if you find a sharper articulation after launch — you can update it at /config anytime
